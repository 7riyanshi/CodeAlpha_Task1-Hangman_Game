# Hangman Game (Python)

## Description
This is a console-based Hangman game developed using Python.
The player has to guess a hidden word by entering one letter at a time.
The player is given a limited number of incorrect attempts.

This project was created as part of the Python Programming Internship
at CodeAlpha.

## Features
- Random word selection
- Letter-by-letter guessing
- Input validation (only single alphabet letters allowed)
- Tracks correct and wrong guesses
- Limited number of attempts
- Clear win and lose conditions

## Technologies Used
- Python 3

## How to Run the Program
1. Make sure Python 3 is installed on your system
2. Open a terminal or command prompt
3. Navigate to the project folder
4. Run the following command:

```bash
python hangman_game.py
